# PDDBot (Telegram Bot)

Телеграм бот для изучения правил дорожного движения(ПДД). 

## Быстрый старт 

### Предварительные требования
- Windows 10/11
- .NET 6.0+
- .NET Framework 4.8+
- Visual Studio 2022
- Telegram WEB/Desktop
- Telegram Bot token

### Установка и запуск 
1. Клонируйте репозиторий
   ```bash
   git clone https://github.com/BanCoder/PDDBot.git
   ```
2. Измените содержимое файла конфигурации (`appsettings.json`)
   ```json
    {
	    "TgSettings": 
        {
		    "token": "your token"
	    }
    }
   ```

>***Важно!*** Для того, чтобы получить необходимый токен, найдите в Telegram специального бота (например: [BotFather](https://t.me/BotFather)), он создаст необходимый токен и сгенерирует `"Что может этот бот? `

3. Запустите программу в Visual Studio 2022
   
### Как использовать бота? 
- Найдите бота в Telegram по имени.
- Отправьте команду /start для начала работы.
- Выбирите тему, которую хотите начать изучать
- Выбирите подтему, которую хотите изучить
- Получите материал в виде сообщения и читайте материал 
- Если хотите вернуться назад, нажмите на соответсвующую кнопку `"Назад"`

### Архитектура 
- Консольное приложение (Майкрософт)
- Telegram.Bot - библиотека для работы с Telegram Bot API
